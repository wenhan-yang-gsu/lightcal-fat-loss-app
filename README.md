# LightCal Fat Loss Coach

LightCal is a free, browser-based calorie deficit and meal logging app. It helps people estimate meals, adjust portions, calculate daily energy needs, and track whether they are staying within a fat-loss target.

## Features

- Meal logging with a local food library
- Free meal templates for common eating scenarios
- Photo upload for visual reference while logging
- Portion adjustment with quick controls
- BMR and daily energy expenditure estimates
- Calorie deficit dashboard
- Macro totals for protein, carbs, and fat
- Multi-language interface: English, Chinese, Japanese, and Spanish
- Local browser storage, with no account required

## Use It Locally

Open `index.html` in a browser.

## Privacy

LightCal stores entries in the user's own browser through local storage. The app does not require sign-in and does not send meal records anywhere else.

## Notes

Calorie and portion values are estimates. Users should adjust portions based on their actual meal size and nutrition labels when available.
